# 让AI学会算数（失败项目）

一个实验性项目，尝试使用LSTM神经网络让AI学习基本的四则运算（加减乘除），但效果极差，被标记为失败项目。

## 项目说明

这是一个使用PyTorch实现的实验性项目，使用LSTM网络来预测四则运算的结果。项目包含训练脚本和交互式推理脚本。

### 技术特点

- 使用LSTM网络进行序列建模
- 支持四种基本运算：加、减、乘、除
- 动态难度调整机制
- 回归预测而非分类

### 项目状态

**实验性项目，效果极差**，可能存在的问题：
- 网络结构不适合数值计算
- 训练数据分布或方式有问题
- 损失函数选择不当
- 数值精度问题

### 文件说明

- `math.py`：训练脚本，包含模型定义和训练循环
- `talk.py`：交互式推理脚本，加载训练好的模型进行预测

### 使用方法

1. 先运行训练脚本：
```bash
python math.py
```

2. 然后运行交互式推理脚本：
```bash
python talk.py
```
