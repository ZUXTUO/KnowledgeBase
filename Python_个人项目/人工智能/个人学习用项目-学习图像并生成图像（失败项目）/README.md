# 学习图像并生成图像（失败项目）

一个基于GAN的图像生成项目，尝试使用生成对抗网络学习图像数据并生成新图像，但项目未能达到预期效果。

## 项目说明

这是一个使用PyTorch实现的生成对抗网络（GAN）项目，旨在从图像数据集中学习并生成新的图像。项目使用了改进的GAN架构，包含生成器和判别器两个网络。

### 技术特点

- 使用PyTorch框架实现
- 生成器：采用反卷积层进行上采样
- 判别器：使用谱归一化（spectral normalization）
- 支持512x512分辨率图像
- 训练过程中保存样本图像和模型检查点

### 项目状态

此项目被标记为"失败项目"，可能存在的问题：
- 训练不稳定
- 生成图像质量不佳
- 模式崩塌问题
- 收敛困难

### 文件说明

- `train.py`：主要训练脚本，包含生成器和判别器的定义及训练循环

### 使用方法

准备图像数据集并放置在`data`目录下，然后运行：
```bash
python train.py
```
