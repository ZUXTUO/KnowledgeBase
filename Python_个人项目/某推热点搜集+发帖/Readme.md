# X（推特）热点搜集+发帖工具

一个自动化工具，用于搜集X（推特）平台的热点话题标签，并自动发布包含这些热点标签的推文。

## 功能特性

- **热点标签搜集**：自动访问X平台的趋势页面，提取当前热门话题标签
- **自动发帖**：根据搜集到的热点标签，自动生成并发布推文
- **定时循环**：支持定时重复执行搜集和发帖任务
- **智能标签过滤**：自动过滤异常字符和过长标签，确保标签质量
- **浏览器自动化**：使用Selenium进行浏览器操作，模拟真实用户行为

## 技术栈

- **编程语言**：Python
- **浏览器自动化**：Selenium WebDriver
- **网页元素定位**：CSS选择器和XPath
- **等待机制**：显式等待和隐式等待

## 使用方法

1. **安装依赖**：
   ```bash
   pip install selenium
   ```

2. **安装ChromeDriver**：
   - 下载与您的Chrome浏览器版本兼容的ChromeDriver
   - 将ChromeDriver添加到系统PATH环境变量中

3. **运行脚本**：
   ```bash
   python x.py
   ```

## 配置选项

- `TRENDING_URL`：趋势页面URL（默认为 `https://x.com/explore/tabs/trending`）
- `COMPOSE_URL`：发帖页面URL（默认为 `https://x.com/compose/post`）
- `INTERVAL_MINUTES`：执行间隔时间（默认为1分钟）

## 工作流程

1. 启动Chrome浏览器（使用临时用户数据目录避免会话冲突）
2. 导航到X平台的趋势页面
3. 搜集当前热门话题标签（以#开头的标签）
4. 过滤异常字符和过长标签
5. 导航到发帖页面
6. 从搜集到的标签中随机选择一个，生成推文内容
7. 自动发布推文
8. 等待指定时间后重复整个流程

## 注意事项

- 需要有效的X平台账户才能发布推文
- 程序会自动创建临时浏览器配置文件，避免会话冲突
- 请遵守X平台的使用条款和发帖频率限制
- 可能需要处理登录验证等安全措施
- 网站结构变化可能导致元素定位失败，需要相应调整选择器

## 应用场景

- 社交媒体营销自动化
- 热点话题跟踪
- 内容推广和曝光
- 自动化社交媒体管理

## 风险提示

- 频繁自动发帖可能导致账户被限制或封禁
- 请合理设置发帖频率，避免被识别为垃圾信息
- 使用前请确保了解并遵守X平台的相关规定
