# 无缝平铺柏林噪音生成器

一个使用Python实现的可无缝平铺的柏林噪音（Perlin Noise）生成工具，能够生成用于纹理制作、程序化内容生成等应用的高质量噪声图像。

## 功能特点

- **无缝平铺**：生成的噪声图像可以在水平和垂直方向无缝拼接
- **多种噪声类型**：支持单层Perlin噪声和多层分形噪声
- **参数可调**：可调整网格分辨率、八度数、持续性等参数
- **可视化输出**：生成灰度图像并保存为PNG格式
- **效果验证**：提供2x2拼接图验证无缝效果

## 技术原理

- **Perlin噪声算法**：实现经典的Perlin噪声生成算法
- **梯度向量处理**：通过强制边缘梯度一致实现无缝平铺
- **分形噪声**：叠加多个八度的噪声以增加细节
- **插值函数**：使用高阶多项式插值保证平滑过渡

```bash
pip install numpy matplotlib
```

## 使用方法

```bash
python3 perlin_noise.py
```

## 输出文件

运行后会生成三个图像文件：

1. `seamless_perlin.png` - 单层Perlin噪声
2. `seamless_fractal.png` - 多层分形噪声（细节更丰富）
3. `tiling_verification.png` - 2x2拼接验证图

## 参数说明

- `img_size`：输出图像尺寸（默认512x512）
- `grid_res`：基础网格大小（默认4x4，数值越小生成的云团越大）
- `octaves`：分形噪声的八度数（默认4）
- `persistence`：噪声层的衰减系数（默认0.5）
- `lacunarity`：频率递增系数（默认2）

## 自定义参数

可以修改代码中的以下参数来自定义生成的噪声：

- `img_size`：调整输出图像大小
- `grid_res`：调整基础网格大小，控制生成图案的尺度
- `octaves`：调整分形噪声的层数，影响细节丰富程度
- `persistence`：调整每层噪声的强度衰减

## 算法说明

1. **梯度生成**：在网格点上生成随机梯度向量
2. **边缘处理**：强制最后一行/列等于第一行/列，确保无缝
3. **插值计算**：使用平滑插值函数计算网格内部的噪声值
4. **多层叠加**：分形噪声通过叠加多个不同频率和振幅的噪声层生成

## 验证无缝效果

程序会生成一个2x2的拼接验证图，如果在拼接边界处没有明显接缝，则说明无缝算法成功。这是检验生成的噪声是否真正可平铺的重要步骤。
