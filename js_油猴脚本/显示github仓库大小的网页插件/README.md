# GitHub仓库大小检测按钮

一个油猴脚本，用于在GitHub仓库页面显示仓库大小信息（以MB为单位）。

## 功能特性

- **实时显示**：在GitHub仓库页面上显示仓库的大小信息
- **美观界面**：使用红色按钮设计，包含硬盘图标，易于识别
- **自动获取**：通过GitHub API自动获取仓库大小数据
- **动态定位**：按钮固定在页面右上角，不影响正常浏览
- **悬停效果**：按钮有悬停动画效果，提升用户体验

## 安装方法

1. 安装油猴浏览器扩展（如Tampermonkey或Greasemonkey）
2. 点击安装链接：[GitHub仓库大小检测按钮-1.3.user.js](./GitHub仓库大小检测按钮-1.3.user.js)
3. 确认安装脚本

## 使用方法

安装脚本后，访问任何GitHub仓库页面，脚本会自动检测并显示仓库大小。仓库大小信息将以MB为单位显示在页面右上角的红色按钮上。

## 技术细节

- 使用GitHub API获取仓库信息
- 通过页面URL解析仓库所有者和名称
- 自动计算仓库大小（KB转MB）
- 动态创建带有样式的按钮元素

## 版本信息

当前版本：1.3

## 注意事项

- 需要网络连接以访问GitHub API
- 遵循GitHub API的使用限制
- 仅在仓库页面显示，不会在其他页面出现
